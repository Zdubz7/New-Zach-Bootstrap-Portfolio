Coin Compare

Coin Compare is a project I worked on with three other developers. Coin Compare is an application that was created for people who have little to no knowledge on crypto currency exchanging but would like to get into it. This project was created in Visual Studio Code in HTML, CSS, and Javascript as well as AJAX to call API's from Coinlib for back end storage and JSON was also used. Firebase was also used in the creation of this application to store user input data on the contact page as well as the blog page. This project was created on GitHub in a repository and forked between three other people. My Role in the creation of this project was the UI/UX graphics and visuals that you see. I created the graphics using a combination of Illustrator, Photoshop, After Effects, and Premiere.

Giftastic Animals

Giftastic Animals is an application I created using HTML, CSS, Javascript, and AJAX to call an API key that generates Gif's of animals at random that are displayed and stored. Once the Gif's are displayed they are static until clicked, when clicked they play on a loop until clicked again and they stop. The user can also type in an animal in the search bar and when submit is clicked that animal will appear as a clickable button to reveal new GIF's. All of the UX / UI design was also done by myself and this project was created and launched in a GitHub repository.

Rock Paper Scissors

For this project I created a rock paper scissors game against the computer except the game has a non-traditional feel. The user either selects The Rock, Edward Scissors, or Benjamin paper and then the computer creates a response to the user's pick, then the score is recorded. This project was coded in HTML, CSS, and Javascript and the repository for the project was created in Github. All of the UI / UX design was done myself using After Effects, Premiere, and photoshop.

Train Schedule

Train Schedule is an app. that I coded in HTML, Javascript, Jquery, JSON, AJAX, and used Bootstrap for styling as well as Firebase for backend storage. This app allows the user to input Data that logs where trains are coming from what time they are going to get there etc. etc. and that data is stored in firebase and will stay there until deleted in firebase.

Lord Of The Rings Hangman

Lord of The Rings Hangman is a game I designed in HTML, CSS, and Javascript. This game displays blanks which the user will guess the character using the hint provided and the alphabet pad to input the letters. Everytime a character is guessed, if correct the letter goes in the space, if the guess is incorrect, the canvas will start drawing limbs of the hangman. When the user gets to 0 lives the game is over, if the user guesses correct or incorrect the score will either be tallied in wins or losses.
