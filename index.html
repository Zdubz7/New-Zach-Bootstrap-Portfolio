About Zach Wiest

Currently I am a Responsive UI | UX developer who is ambitious, persistent, and dedicated at honing my craft. I enjoy coding apps, webpages, games, creating databases, and enlightening myself in all subject matter that encompasses development and the digital world. I have advanced skills in the Adobe Suite as well as various coding languages. I'm also a photographer, video editor, and motion graphic ui designer. My goal in life is to be the best developer I can while crafting extraordinary web and various digital solutions for my employer. I embrace new challenges and I'm always willing to learn whatever I have to, to be a team player. I don't look at failure as being a burden or let it get me down, but rather a learning experience that I can learn and grow from. Its hard for me to believe skies the limit when we have footprints on the moon, in life you are only in a race with yourself and you can only achieve as much as you believe you can. Be somebody.

In my freetime I like to go on adventures of all different kinds and while I'm adventuring I like to take photographs of scenery as well as shoot and edit videos. I also like to paint in my freetime in acrylic and oil as well. Other than that most of my time is spent learning coding and learning new design techniques. I like to create as much and use as much or my own original material as possible so that all of my work is always fresh and original. Coin Compare

Coin Compare is a project I worked on with three other developers. Coin Compare is an application that was created for people who have little to no knowledge on crypto currency exchanging but would like to get into it. This project was created in Visual Studio Code in HTML, CSS, and Javascript as well as AJAX to call API's from Coinlib for back end storage and JSON was also used. Firebase was also used in the creation of this application to store user input data on the contact page as well as the blog page. This project was created on GitHub in a repository and forked between three other people. My Role in the creation of this project was the UI/UX graphics and visuals that you see. I created the graphics using a combination of Illustrator, Photoshop, After Effects, and Premiere. The raw files for this application can be found at https://github.com/Zdubz7/Coin-Compare.

Giftastic Animals

Giftastic Animals is an application I created using HTML, CSS, Javascript, and AJAX to call an API key that generates Gif's of animals at random that are displayed and stored. Node.JS Lord Of The Rings Constructor Hangman

Node.JS Lord of The Rings Comnstructor Hangman is a game I designed using Javascript and Node.js. The packages that this game was created in are the Inquirer package as well as the Is-Letter npm package. When you open the github repository project containing the code you need to clone the repository to your terminal and run npm install to install Node.JS before running the program. Once installed the user will run node hangman.js in the terminal to begin the game and will be prompted with blanks. The user is given 10 guesses per prompt and will then type in one single letter at a time to guess the Lord Of The Rings Characters. Everytime a letter is guessed wrong, a piece of the hangman will be drawn in the terminal window. If the user completely fails to guess the character correctly, the user will lose and have to input node hangman.js to run the game again. The raw files for this application can be found at https://github.com/Zdubz7/Constructor-Word-Guess.

New York Times MongoDB Scraper

The New York Times MongoDB Scraper displays the most recent articles being created by the New York Times. This was made possible because of the MongoDB database I created as well as using Heroku MongoDB for hosting and deployment as well as github for original deployment. These are all Node.Js packages that need to be installed in your terminal. This application's files can also be found on my github page https://github.com/Zdubz7/All-The-News-Thats-Fit-To-Scrape.

Liri Node Application

This Liri Node Application is an application that I designed using Javascript and Node.JS. Whats happening in this app is Twitter, OMDB, and Spotify API keys are being used to pull information from a fake Denzel Washington Twitter page that I made to display 20 of my latest tweets, OMDB is pulling information from Rotten Tomatoes to display the movie name, year, and rating, as well as 10 songs are being pulled from spotify to create a comparison of song titles that are alike. All of this is done within a terminal after you download all of the Node.Js packages that I have provided in the READ ME page within my github repository. I have created custom error messages that are displayed in the users terminal when a user doesn't correctly input a Node.JS value for the Liri Application Command Line. All of the information that is being pulled from the API Keys are displayed in a log.txt file within my Liri Node Application repository on Github. All the raw files of this application can be found at https://github.com/Zdubz7/liri-node-app.

Dragon Ball Z React.Js Clicky Game

Dragon Ball Z React.Js Clicky Game is a game that tests the users memorization of the different Dragon Ball Z characters provided. The user must select each character without guessing the same character twice. If the user gets all 12 characters guessed correctly the user wins, if the user guesses the same character twice they lose and the score is calculated. Each time the character guesses a correct answer the score is incremented. This game was created using Javascript, Css, React.Js, and Node.Js.
